# ProxyPool

一个golang编写的简单代理池

## Feature
- 在网络中获取免费代理
- 自动认证代理
- 提供一个Web API供调用
- 快得像闪电

## 构建使用
你必须保证你的计算机上安装了golang。

```
cd source
git clone https://github.com/YHaoNan/ProxyPool.git
cd ProxyPool
go build
./HTTPProxyPool
```

然后，你就可以访问`http://localhost:4000`去获取代理列表。
